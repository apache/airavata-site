Mailing Lists

We welcome you to join mailing lists and let us know about your thoughts or ideas about Apache Airavata.

Developer Mailing List

Apache Airavata Developer Mailing List is where the community hangs out and discussion about Airavata development happens. This list is used to coordinate activities and ensure we are all pulling in the same direction.

Subscribe: dev-subscribe@airavata.apache.org
Post (after subscription): dev@airavata.apache.org
Unsubscribe: dev-unsubscribe@airavata.apache.org
Archives:
  • Apache Lists Archive
  • Markmail Archive
  • Incubator Archive at Apache (May 2011 to Septemeber 2012)

Custos Mailing List

Apache Airavata Custos Mailing List is where you can ask questions about Airavata Custos Security Projects. All the developers are subscribed to the list and you may ask questions on how to use or contribute to Custos Security framework.

Subscribe: custos-subscribe@airavata.apache.org
Post (after subscription): custos@airavata.apache.org
Unsubscribe: custos-unsubscribe@airavata.apache.org
Archives:
  • Apache Lists Archive

User Mailing List

Apache Airavata User Mailing List is where you can ask questions about Airavata. All the developers are subscribed to the list and you may ask questions on how to integrate Airavata into existing frameworks, or how to use Airavata out of the box. If you have a use case you are intending to use Airavata, please convey it to this list.

Subscribe: users-subscribe@airavata.apache.org
Post (after subscription): users@airavata.apache.org
Unsubscribe: users-unsubscribe@airavata.apache.org
Archives:
  • Apache Lists Archive
  • Markmail Archive

Architecture Mailing List

Apache Airavata Architecture Mailing List is have the architecture discussion focused and managable for casual contributors, airavata conducts most of the architecture brainstorming on a architecture mailing list. This list is used to coordinate activities and ensure we are all pulling in the same direction.

Subscribe: architecture-subscribe@airavata.apache.org
Post (after subscription): architecture@airavata.apache.org
Unsubscribe: architecture-unsubscribe@airavata.apache.org
Archives:
  • https://lists.apache.org/list.html?architecture@airavata.apache.org">Apache</a>
  • Markmail Archive

Issues Mailing List

To keep the dev list traffic down from JIRA issues, comments, GitHUB Pull Requests and comments, these are sent to the Apache Airavata Issues Mailing List. Developers are encouraged to subscribe to this list to follow all development roadmaps and code reviews.

Subscribe: issues-subscribe@airavata.apache.org
Unsubscribe: issues-unsubscribe@airavata.apache.org
Archives:
  • Apache Lists Archive
  • Markmail Archive

Note: This is a notification only mailing list and all queries have to directed to user/dev/architecture lists as appropriate.

Commit Mailing List

Apache Airavata Commit Mailing List is where the community receives automated notifications of any changes to the airavata code and documentation.

Subscribe: commits-subscribe@airavata.apache.org
Unsubscribe: commmits-unsubscribe@airavata.apache.org
Archives:
  • Apache Lists Archive
  • Markmail Archive
  • Incubator Archive at Apache (May 2011 to Septemeber 2012)

Note: This is a notification only mailing list and all queries have to directed to user/dev/architecture lists as appropriate.
